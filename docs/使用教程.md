## 常用控制台命令（都在微信开发者工具 Console 里执行）

```javascript
// 1) 初始化（只一次）
wx.cloud.callFunction({ name: 'init' }).then(console.log).catch(console.error)

// 2) 手动检查异常考勤（测试）
wx.cloud.callFunction({ name: 'attendance', data: { action: 'checkAbnormal' } }).then(console.log).catch(console.error)

// 3) 快速检查 init 云函数是否可调用
wx.cloud.callFunction({ name: 'init' }).then(res => console.log('init ok', res)).catch(err => console.error('init fail', err))
```

## 报错时先看这 3 项
1. 当前云环境是否选对（开发者工具云开发面板右上角环境）。
2. `init` 云函数是否部署到同一个环境。
3. 小程序是否重新编译过（建议“清缓存并编译”一次）。

## 给最终用户的日常操作（超简版）
- 员工：登录 → 打卡 → 报工。
- 质检：登录 → 待检 → 提交质检。
- 管理员：查看报表/薪资 → 导出。
