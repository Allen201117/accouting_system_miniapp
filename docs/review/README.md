# 微信小程序提审说明（3.3/3.4 对应）

## 1. 审核体验路径（无账号）
1. 进入小程序登录页。
2. 阅读并同意《隐私政策》《用户服务协议》。
3. 点击 `体验模式（免登录浏览）`。
4. 在体验模式页查看核心流程说明、示例看板数据、协议入口。
5. 返回登录页，可继续使用 `审核快速登录` 进入真实流程体验。

## 2. 审核体验路径（快速登录）
1. 进入登录页并完成协议确认。
2. 点击 `审核快速登录`（由云端开关 `factory_settings.review_mode_enabled` 控制）。
3. 进入老板端首页后，建议按以下路径体验：
   - 首页 -> 考勤码：生成二维码、作废二维码、查看二维码ID。
   - 首页 -> 数据中心：查看考勤/报工/薪资统计。
   - 首页 -> 设置：查看隐私政策与用户服务协议入口。

## 3. 如需在提审表单填写账号密码（模板）
- 体验账号：`[请填写云端配置的 review_user_name]`
- 体验手机号：`[请填写云端配置的 review_user_phone]`
- 登录方式：小程序内 `审核快速登录`，无需手动输入密码。
- 权限说明：只读/受限账号，不会污染真实业务数据。

> 配置位置：`factory_settings/main`
> - `review_mode_enabled: true`
> - `review_user_name: 审核账号姓名`
> - `review_user_phone: 审核账号手机号`

## 4. 提审录屏建议（30-60 秒）
1. 展示登录页首次弹出隐私协议确认弹窗，点击协议页面并返回。
2. 勾选同意后，点击 `审核快速登录`。
3. 进入老板首页，点击 `考勤码`，演示生成二维码、显示二维码ID。
4. 点击 `作废当前二维码`，展示作废后提示。
5. 返回首页进入 `数据中心`，展示考勤/报工/薪资三个 Tab。
6. 打开 `设置` 页面，展示隐私政策和用户服务协议入口。
