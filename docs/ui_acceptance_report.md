## 问题 1 验收：工序分配页"选人列表看不出选中了谁"

### 1.1 实现内容

| 需求项 | 实现方式 |
|--------|---------|
| 列表项选中态 | 左侧圆形 checkbox（选中蓝底白勾 / 未选空心圆）；选中行背景色变为浅蓝 `#F0F9FF`；右侧追加蓝色 ✔ 图标 |
| 员工信息 | 主行：姓名（36rpx 加粗），副行：角色（员工/质检员） + 入厂时间 |
| 点击整行切换 | `bindtap="toggleAssignUser"` 绑定在 `.assign-emp-row` 整行 |
| 顶部已选预览区 | 横向可滚动 `<scroll-view scroll-x>` Tag/Chip，显示姓名 + ✕ 可删除 |
| 已选计数 | 预览区标题 "已选 X 人"，底部操作栏也有 "已选 X 人" |
| 清空按钮 | 已选区右上角红色「清空」按钮 |
| 底部固定操作栏 | sticky 定位在全屏面板底部，左侧 "已选 X 人"，右侧 "取消" + "保存分配" |
| 保存按钮禁用逻辑 | `assignHasChange` 计算 diff（`_arraysChanged`）：无变更时灰色禁用显示 "无变更"；有变更时蓝色可点 |
| 保存前确认弹窗 | 标题 "确认保存工序分配"，显示工序名 + 前 5 人姓名 tag + "等 X 人" 提示，双按钮 取消/确认保存 |
| 空选清空提示 | 确认弹窗内：若已选为空，红色提示 "将清空该工序的所有分配" |

### 1.2 验证步骤

1. 进入订单详情 → 点击任一工序"分配员工"/"编辑分配"按钮
2. **全屏分配面板打开**，顶部显示工序名
3. **选择员工**：点击行 → 左侧 checkbox 变蓝勾 + 行背景变浅蓝 + 右侧出现 ✔
4. **顶部已选区**：看到 Tag（姓名 + ✕），横向可滚动
5. **点击 Tag 的 ✕**：该员工取消选中，checkbox 恢复空心
6. **点击清空**：所有选中状态清除
7. **无变更**：底部按钮显示"无变更"且灰色不可点
8. **有变更**：底部按钮显示"保存分配"可点
9. **点击保存**：弹出确认弹窗 → 显示工序名 + 员工名 tag 列表
10. **确认保存**：写库成功 → toast "分配成功" → 自动关闭面板 → 订单详情自动刷新

## 问题 2 验收：订单详情页"工序分配展示不明显"

### 2.1 实现内容 — 三区块卡片化布局

| 区块 | 内容 |
|------|------|
| **区块1：订单概览卡** | 订单名（大字）+ 状态标签（进行中/已完成）+ 2×2 meta grid（总数量、开始日期、截止日期） |
| **区块2：分配总览卡** | 三个统计数字（工序总数 / 已分配 / 未分配）+ 三 Tab 筛选（全部 / 未分配 / 已分配） |
| **区块3：工序列表卡** | 工序卡片列表，每卡包含工序名 + 备注 + 单价 + 分配状态展示 + 操作按钮 |

### 2.2 工序卡片强分配展示

| 状态 | 展示 |
|------|------|
| **已分配** | 绿色左边框 `#52C41A`，绿色 badge "已分配 X 人" + 蓝色姓名 Tag（张三、李四…） |
| **未分配** | 红色左边框 `#FF4D4F`，暖黄底色 `#FFFBF0`，红色 badge "未分配" + 红色按钮 "去分配 →" |

### 2.3 筛选功能

- **全部** Tab：显示所有工序
- **未分配** Tab：仅显示 `assigned_user_ids.length === 0` 的工序，右上角红色数字 badge
- **已分配** Tab：仅显示已分配工序
- 空筛选结果有对应空状态文案

### 2.4 联动验证

1. 从订单详情进入分配面板 → 保存成功 → 自动关闭面板 → `onShow()` 中 `loadOrderDetail()` 触发刷新
2. 工序卡片的分配 tag 即时更新
3. 总览卡的已分配/未分配数字即时更新
4. Tab 筛选器的 badge 数字即时更新

### 2.5 验证步骤

1. 进入有 3 个工序（2 已分配 + 1 未分配）的订单
2. **总览卡**：工序总数 3，已分配 2（绿色），未分配 1（红色）
3. **全部 Tab**：显示 3 个工序卡片
4. **未分配 Tab**：显示 1 个红边框暖黄底色卡片，有 "未分配" 红标签 + "去分配 →" 按钮
5. **已分配 Tab**：显示 2 个绿边框卡片，有 "已分配 X 人" 绿标签 + 姓名蓝 tag
6. 点击 "去分配 →"：进入全屏分配面板（问题1），选人保存后返回 → 未分配数变 0
